ipaddr.ca the one with the silly bird


This website will tell you your IPv4 and IPv6 addresses, as well as PTR if it exists.
This website is usable in shell scripts too.

Scripting Usage

This website can also be shell scripted using curl and httpie

Get IPv4 Address in Bash

IP4=`curl -s 4.ipaddr.ca/ip`

Get IPv6 Address in Bash

IP6=`curl -s 6.ipaddr.ca/ip`

Get IPv4 PTR Address in Bash

PTR4=`curl -s 4.ipaddr.ca/ptr`

Get IPv6 PTR Address in Bash

PTR6=`curl -s 6.ipaddr.ca/ptr`

Single Stacked Hosts

Single stacked hosts which don't have IPv6 access can simply request the following:

IP=`curl -s ipaddr.ca/ip` or
PTR=`curl -s ipaddr.ca/ptr`

Advanced Scripting Usage

This website can also be shell scripted using curl:
IP6=`curl -s 6.ipaddr.ca/ip` or
IP4=`curl -s 4.ipaddr.ca/ip` or
PTR6=`curl -s 6.ipaddr.ca/ptr` or
PTR4=`curl -s 4.ipaddr.ca/ptr`
